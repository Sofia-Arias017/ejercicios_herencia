# Ejercicios de Herencia en Java

Este repositorio contiene 10 ejercicios sobre herencia, clases, metodos y polimorfismo en Java.  
Cada ejercicio esta organizado en su propia carpeta y contiene las clases necesarias segun el enunciado.

### Ejercicio 1
Crear una clase `Vehicle` con el metodo `move()`.  
Crear la subclase `Car` que hereda de Vehicle y agrega el metodo `honk()`.

### Ejercicio 2
Crear la clase `Person` con los atributos `name` y `age`.  
Crear la clase `Student` que hereda de Person y agrega `grade` y el metodo `study()`.

### Ejercicio 3
Crear la clase `Animal` con el metodo `makeSound()`.  
Crear `Dog` y `Cat` que sobrescriban el metodo diciendo "Woof" y "Meow".

### Ejercicio 4
Crear la clase `Employee` con los atributos `name` y `salary`.  
Crear la clase `Manager` que hereda y agrega el atributo `department`.

### Ejercicio 5
Crear una clase abstracta `Shape` con el metodo `calculateArea()`.  
Crear `Circle` y `Rectangle` que implementen ese metodo.

### Ejercicio 6
Crear la clase `Bird` con el metodo `fly()`.  
Crear `Eagle` que sobrescriba el metodo y llame a `super.fly()`.

### Ejercicio 7
Crear la clase `Device` con un constructor que imprima "Device created".  
Crear la clase `Phone` que hereda y en su constructor imprima "Phone ready".

### Ejercicio 8
Crear la clase `Account` con el atributo `balance` y los metodos `deposit()` y `withdraw()`.  
Crear `SavingsAccount` que agrega el metodo `addInterest()`.

### Ejercicio 9
Crear la clase `Vehicle` y las subclases `Car`, `Bike` y `Truck`.  
Cada una debe sobrescribir el metodo `describe()`.

### Ejercicio 10
Crear un `ArrayList<Animal>` y agregar instancias de `Dog`, `Cat` y `Bird`.  
Recorrer la lista y llamar al metodo `makeSound()`.
